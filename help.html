RPG Game Help

Objective
The player is in a maze
Menu Navigation
Title Screen:
  • L-CLICK to execute the selected button
  • Singleplayer: creates a game where there is only one player(you)
  • Join Game: opens a new interface where the player can join a server
  • Create Game: creates a new multiplayer game where the player is the host
  • Create Character: opens a new interface for the player to create a new character
  • Quit: ends the game, closes the window
Join Game Screen:
  • If the GNS is online and there is a server online, the join game screen will show once create game has been pressed. Otherwise, an error will show.
  • All servers currently online will be listed
  • L-CLICK to join the selected server
Create Game:
  • To create a multiplayer game the GNS has to be running. Otherwise, an error will show.
Character Creator Screen:
  • a-z to enter the name for the new character
  • L-CLICK to select one of the classes, represented with a green border once selected
  • L-CLICK to confirm the character creation
Controls
Movement:
  • W to go forwards
  • A to go left
  • S to go backwards
  • D to go right
User Interface:
  • ESC to open the pause menu
  • SCROLL-WHEEL-UP/DOWN to change the selected item in the hotbar
  • E to open the inventory
  • 1 - 5 to swap items from the inventory to hotbar, done while hovering over the desired item in the inventory
  • R-CLICK to swap armor items from the inventory to equipment, done while hovering over the desired item in the inventory
  • K to open the skill tree
Item Usage:
  • Moving the mouse around the board will indicate where the player is looking
  • L-CLICK will use the selected item (weapon/consumable) in the hotbar
